Apple Watch Music App UI

Completed 2017

This is a class project that I decided to enhance further. The goal was to create music app for the small-screened Apple Watch. The trickiest thing about this project was following the strict guidelines on the Apple developer website.

I had to learn how to fit every necessary button so that it could be seen but also touched with a finger, but also use the required Apple fonts, buttons and layouts. This project taught me that not all rules can be broken in design when trying to get projects featured to the public.
